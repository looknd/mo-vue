# Mo-vue

> vue 2.0 ui components


## Demo

[demo && docs](https://s-mohan.github.io/demo/mo-vue/)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```

## 目录

``` html 
src : 文档和实例
packages : MoUi
```
